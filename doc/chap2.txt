  [33X[0;0YSemirings  and  various  related  algebraic structures of small order can be
  enumerated  using  the [12XSemirings[112X package. By small order, we mean at most 8,
  but  in  many  cases  even 8 is computationally unfeasible under the current
  implementation.  Each  structure  type  has an associated pair of functions:
  [10XAll<Structure>[110X   for  enumeration  and  [10XNr<Structure>[110X  for  counting.  Here,
  [10X<Structure>[110X  can  be  one  of:  [10XAiSemirings[110X, [10XRingsWithOne[110X, [10XRings[110X, [10XSemirings[110X,
  [10XSemiringsWithOneAndZero[110X,    [10XAiSemiringsWithOneAndZero[110X,    [10XSemiringsWithZero[110X,
  [10XAiSemiringsWithZero[110X, [10XAiSemiringsWithOne[110X, or [10XSemiringsWithOne[110X.[133X
  
  [33X[0;0YFor  a  given [10X<Structure>[110X, the [10XAll<Structure>[110X function returns a list of all
  structures of that type (up to isomorphism or equivalence), each represented
  as  a pair of Cayley tables. For instance, [10XAllSemirings(n)[110X returns a list of
  pairs  [10X[A,  M][110X,  where  [10XA[110X  and [10XM[110X are [23Xn\times n[123X Cayley tables as described in
  Section  [14X1.1[114X. The corresponding [10XNr<Structure>[110X function returns the number of
  such  structures without storing them, and so is both faster and more memory
  efficient.[133X

  [1X2.1-3 AllSemiringsWithX[101X
  
  [33X[1;0Y[29X[2XAllSemiringsWithX[102X( [3Xn[103X, [3XstructA[103X, [3XstructM[103X[, [3XU2E[103X] ) [32X function[133X

  [33X[0;0YThese  functions  allow  you  to  enumerate/count the number of semirings of
